## 简介

提交万物迁移-服装模式任务，用于实现虚拟试装功能。

## 认证

<ParamField header="Authorization" type="string" required>
  Bearer Token，如 `Bearer eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9...`
  获取方法：依照左侧最上面认证模块的post接口获取，`https://docs.openai-nebula.com/cn/nebula-lab-v4/endpoint/get-token`
</ParamField>

### 示例代码

<Tabs>
  <Tab title="cURL示例">
    ```bash
    curl -X POST "https://ai-nebula.com/prod-api/nebula/v4/api/dressing2Submitv4" \
      -H "Content-Type: application/json" \
      -H "Authorization: Bearer sk-xxxxxxxxxx" \
      -d '{
        "imageUrl": "https://example.com/model.jpg",
        "clothingUrl": "https://example.com/clothing.jpg"
      }'
    ```
  </Tab>
  <Tab title="Python示例">
    ```python
    import requests
    import json

    url = "https://ai-nebula.com/prod-api/nebula/v4/api/dressing2Submitv4"
    headers = {
        "Content-Type": "application/json",
        "Authorization": "Bearer sk-xxxxxxxxxx"
    }
    data = {
        "imageUrl": "https://example.com/model.jpg",
        "clothingUrl": "https://example.com/clothing.jpg"
    }

    response = requests.post(url, headers=headers, data=json.dumps(data))
    print(response.json())
    ```
