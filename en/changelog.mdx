## 📅 March 2026

### 🚀 GPT-5.4 Series Released (03/05)

OpenAI's new flagship-for-work series is now available across ChatGPT, API, and Codex:

- **Official positioning**: GPT‑5.4 is presented as a frontier model for professional work, with `gpt-5.4-pro` for maximum performance
- **Native computer use**: First general-purpose OpenAI model with native computer-use support for agent workflows
- **Long context**: Supports up to 1M token context in Codex/API scenarios
- **Efficiency upgrade**: Officially positioned as more token-efficient than GPT‑5.2 for many complex tasks

### 🎨 Doubao Seedream 5.0 Released (02/10)

ByteDance Seed's latest image model is now available:

- **Model name**: `doubao-seedream-5-0-260128`
- **Official focus**: Seedream 5.0 Lite emphasizes "deeper thinking, more accurate generation"
- **Core capabilities**: Better multimodal understanding/reasoning, stronger consistency and instruction following
- **Official feature**: Real-time search enhancement (toggleable) for time-sensitive creation workflows

### 🧠 Claude 4.6 Series Update (02/05)

Anthropic's new generation is now supported:

- **Flagship pair**: `claude-sonnet-4-6` + `claude-opus-4-6`
- **Official upgrades**: Improved coding, debugging, code review, and long-running agent performance
- **Long context**: Anthropic officially introduced 1M token context (beta) for Opus-class workflows

### 🌟 Gemini 3.1 Series Released (02/19)

Google's latest Gemini generation is now available:

- **Official positioning**: Gemini 3.1 Pro is announced as a stronger core model for complex tasks (Preview)
- **Capability direction**: Improved reasoning for hard problem-solving and more advanced agentic workflows
- **Availability**: Rolled out via Gemini API, Vertex AI, Gemini app, NotebookLM, and related developer channels
- **Current options**: `gemini-3.1-pro-preview`, `gemini-3.1-flash-lite-preview`, `gemini-3.1-flash-image-preview`

## 📅 December 2025

### 🚀 GPT-5.2 Series Released (12/12)

OpenAI's year-end upgrade for Chat, Reasoning, and Code:

- **Performance**: Improved reasoning chain; more stable code/tool calls; better SWE-bench than 5.1
- **Pricing**: Same as GPT-5.1, keeping strong value
- **Compatibility**: Streaming, tool calling, and thinking/reasoning remain backward-compatible
- **Productivity focus**: Targeted at spreadsheets, decks, long docs, and coding; positioned as a "brainwork partner"
- **GDPval benchmark**: Across 44 professions and 1,320 real tasks, GPT-5.2 matches or beats 14-year experts in 70.9% of cases
- **ARC-AGI 2 leap**: Thinking version scores 52.9% (up from 17.6%), notably ahead of Gemini 3 Pro / Claude peers
- **Single mega-agent**: Lower tool-call latency, smoother multi-tool orchestration
- **Vision + long context**: Industrial-grade vision (e.g., circuit boards, wave simulations) with ultra-long context for multi-file and long-doc analysis
