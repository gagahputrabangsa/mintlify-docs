## Get started in three steps

<Steps>
  <Step title="Clone your repository">
    During onboarding, Mintlify creates a GitHub repository for your docs. Clone it locally to start editing:
    ```bash
    git clone https://github.com/your-org/your-docs
    cd your-docs
    ```
  </Step>
  <Step title="Preview your docs">
    Install the Mintlify CLI and start the local preview server:
    ```bash
    npm i -g mint
    mint dev
    ```
    Your docs are now live at `http://localhost:3000` and update as you save files.
  </Step>
  <Step title="Publish your changes">
    Push to your default branch. The Mintlify GitHub app picks up the changes and deploys them automatically — no CI pipeline needed.
  </Step>
</Steps>
