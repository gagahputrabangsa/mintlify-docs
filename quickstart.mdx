Mintlify connects to your GitHub repository and automatically deploys your docs on every push. This guide walks you through cloning your repository, previewing it locally, and going live for the first time.

## Prerequisites

- A [Mintlify account](https://dashboard.mintlify.com) and an active docs project
- [Node.js v19+](https://nodejs.org) installed
- [Git](https://git-scm.com) installed

## Steps

<Steps>
  <Step title="Clone your docs repository">
    Your docs repository was created automatically during Mintlify onboarding. Find the link in your [dashboard](https://dashboard.mintlify.com).

    ```bash
    git clone https://github.com/your-org/your-docs
    cd your-docs
    ```

    <Tip>If you already have a docs repository, you can connect it from **Dashboard → Settings → GitHub**.</Tip>
  </Step>

  <Step title="Install the Mintlify CLI and start a preview">
    The Mintlify CLI (`mint`) lets you preview your docs locally with hot reload.

    ```bash
    npm i -g mint
    mint dev
    ```

    Open [http://localhost:3000](http://localhost:3000) to see your docs. The preview updates automatically whenever you save a file.
  </Step>

  <Step title="Customize your site name and colors">
    Open `docs.json` in your editor. Change the `name` and `colors.primary` fields to match your brand:

    ```json docs.json
    {
      "name": "Your Product",
      "colors": {
        "primary": "#your-hex-color"
      }
    }
    ```

    Save the file and watch your preview update instantly.
  </Step>

  <Step title="Deploy to production">
    Install the [Mintlify GitHub App](https://dashboard.mintlify.com/settings/organization/github-app) from your dashboard. Once installed, every push to your default branch triggers an automatic deployment — no CI/CD configuration required.

    ```bash
    git add .
    git commit -m "Initial docs setup"
    git push
    ```

    Your changes will be live within moments.
  </Step>
</Steps>
