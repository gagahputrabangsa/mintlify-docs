## 一、产品简介

Dify 是「可视化 + 插件化」的 AI 应用搭建平台，零代码基础也能轻松用大模型，快速搭建专属 AI 工具（比如对话机器人、文本处理工具等），小白也能上手！

通过集成 **Nebula API**，您可以在 Dify 中使用成本仅为官方 1/5 甚至更低的大模型服务，使用效果却几乎无差，高频使用者直接省出奶茶钱～

## 二、集成步骤

### 第一步：准备第三方 API 账号

先注册一个 Nebula API 服务商账号：

👉 https://openai-nebula.com

![注册页面1](/images/3.png)

![注册页面2](/images/4.png)

![注册页面3](/images/5.png)

### 第二步：进入 Dify 模型供应商设置

1. 打开 Dify 平台，登录后找到「设置」页面
2. 左侧菜单中选中「模型供应商」，进入插件安装界面

![模型供应商设置](/images/6.png)

### 第三步：安装兼容插件

1. 下滑页面，找到「OpenAI-API-compatible」插件
2. 这款插件超实用！支持对话、文本嵌入、语音转文字等多种功能，能让 Dify 无缝对接第三方大模型，点击「安装」即可

![安装插件](/images/7.png)

### 第四步：配置关键参数（重点！）

安装完成后，点击「增加模型」，需要从 https://openai-nebula.com 获取 3 个核心参数，一步步来：

![配置参数1](/images/7.png)

![配置参数2](/images/8.png)

![配置参数3](/images/9.png)

#### 1. 获取 API Key（📝 复制粘贴就行）

- 打开 https://openai-nebula.com，点击顶部导航「令牌」
- 复制页面生成的「令牌」，粘贴到 Dify 参数的「API Key」字段

#### 2. 获取模型名称（⚠️ 两个字段要一致）

- 回到 https://openai-nebula.com，进入「账户」页面
- 页面下方蓝色「模型列表」中，选你需要的模型（比如 Gemini、GPT 系列），点击复制名称
- 把复制的名称，分别填入 Dify 的「模型名称」和「API endpoint 中的模型名称」（别填错！）

#### 3. 填写 API 地址（直接抄作业！）

在 Dify 的「API endpoint」字段中填入：

```
https://llm.ai-nebula.com/v1
```

#### 4. 保存配置

所有参数填完，点击「保存」—— 搞定！第三方模型成功接入 Dify～
