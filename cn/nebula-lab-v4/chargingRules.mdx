## 简介

Nebula Lab V4 API 采用**积分制**计费。

## 计费说明

- 积分在**提交任务成功**时扣除（部分视频类接口为预扣费，任务成功后才正式入账）。
- 时长类接口支持格式：`3s`、`5s`、`8s`、`10s`、`12s` 等，积分按**换算后的秒数**计算。
- 积分保留 1 位小数。

---

## 视频生成

### 文生视频

| 接口 | 计费规则 |
|------|----------|
| 文生视频-提交任务 | **每秒 0.6 积分**，按请求参数 `duration` 换算成秒后计算 |
| 文生视频-查询任务 | 不扣费 |

### 图生视频-传统模式

| 接口 | 计费规则 |
|------|----------|
| 图生视频-传统模式-提交任务 | **每秒 0.1 积分**，按 `duration` 换算成秒后计算 |
| 图生视频-传统模式-查询任务 | 不扣费 |

### 图生视频-首尾图模式

| 接口 | 计费规则 |
|------|----------|
| 图生视频-首尾图模式-提交任务 | **每秒 0.4 积分**（等价于 5 秒 2 积分），按 `duration`（秒）计算 |
| 图生视频-首尾图模式-查询任务 | 不扣费 |

### 图生视频-视频换人

| 接口 | 计费规则 |
|------|----------|
| 图生视频-视频换人-提交任务 | **每秒 0.4 积分**，按传入视频 URL 解析得到的视频时长（秒）计算 |
| 图生视频-视频换人-查询任务 | 不扣费 |

## 图片生成（AI 生图）

以下接口均为**按张数**计费，单张 **0.2 积分**。若接口支持数量参数 `n`，则总积分 = **0.2 × n**。

| 接口 | 计费规则 |
|------|----------|
| AI生图-文生图 | 每张 **0.2 积分**（单次 1 张） |
| AI生图-图生图 | 每张 **0.2 积分**（单次 1 张） |
| AI生图-批量生图 | 每张 **0.2 积分**，总积分 = 0.2 × 参数 `n` |
| AI生图-裂变生图 | 每张 **0.2 积分**，总积分 = 0.2 × 参数 `n` |
| AI生图-服装种草 | 每张 **0.2 积分**，默认 1 张，可传 `n` |
| AI生图-智能去印 | 每张 **0.2 积分**，默认 1 张，可传 `n` |
| AI生图-商品提取 | 每张 **0.2 积分**，默认 1 张，可传 `n` |
| AI生图-服装细节 | 每张 **0.2 积分**，默认 1 张，可传 `n` |
| AI生图-高清修图 | 每张 **0.2 积分**，默认 1 张，可传 `n` |

| 万物迁移-查询任务 | 不扣费 |

## 扣费与预扣费说明

- **即时扣费**：多数图片生成、图生视频（传统/首尾图）、万物迁移创意/服装模式等，在提交成功即从账户扣除积分。
- **预扣费**：文生视频、图生视频-视频换人、万物迁移-标准模式、万物迁移-服装模式等，提交时先创建“未支付”积分记录，等任务成功（如查询到成功状态或回调成功）后再标记为已支付；若任务失败，按业务逻辑可能不扣或退还。

调用前请确保账户积分充足，否则将收到余额不足类错误。
