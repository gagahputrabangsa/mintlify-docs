## 💰 定价说明

### 计费方式

- **按量计费**：根据实际使用的 Token 数量计费
- **无最低消费**：充多少用多少，余额永不过期
- **实时扣费**：每次调用后立即从余额扣除费用

### 价格优势

- 官方源头转发，价格略有优势
- 批量使用可联系客服获取更优惠价格

### 成本优化建议

1. **分级使用**：简单任务用便宜模型，复杂任务用高级模型
2. **测试优化**：先用小模型测试，确定需求后再用大模型
3. **批量处理**：大量相似任务可以选择 Mini 版本
4. **缓存复用**：对重复查询结果进行缓存
