## Introduction

Submit Any Shoot dressing mode task, used for virtual try-on.

## Authentication

<ParamField header="Authorization" type="string" required>
  Bearer Token, e.g., `Bearer eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9...`
  How to get: Obtain via the POST interface in the Authentication module at the top left, `https://docs.openai-nebula.com/en/nebula-lab-v4/endpoint/get-token`
</ParamField>

    ```bash
    curl -X POST "https://ai-nebula.com/prod-api/nebula/v4/api/dressing2Submitv4" \

      }'
    ```

  <Tab title="Python Example">
    ```python
    import requests
    import json

    url = "https://ai-nebula.com/prod-api/nebula/v4/api/dressing2Submitv4"
    headers = {
        "Content-Type": "application/json",
        "Authorization": "Bearer sk-xxxxxxxxxx"
    }
    data = {
        "imageUrl": "https://example.com/model.jpg",
        "clothingUrl": "https://example.com/clothing.jpg"
    }

    response = requests.post(url, headers=headers, data=json.dumps(data))
    print(response.json())
    ```
